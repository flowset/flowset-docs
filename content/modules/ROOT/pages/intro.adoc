= Introduction

== What is Flowset?

Flowset is a business process automation platform with its own execution engine and a set of tools covering the entire BPM solution lifecycle.

=== Core Principles

* **Compatibility with Camunda**
  We believe this technology, which has become an industry standard, will last for a long time.

* **Adherence to the BPMN 2.0 Standard**
  Ensures transparency, predictability, and facilitates talent acquisition.

* **Interoperability**
  Flowset is compatible with popular modeling and development tools.

* **Focus on Business Process Automation**
  Flowset covers the full cycle of process automation—from design to execution and monitoring.

* **Modularity and Openness**
  Each component of the platform works independently, and together they deliver maximum value.

=== Platform Components

==== Control
An open-source tool for monitoring business process execution. Similar to Camunda Cockpit but with extended functionality.
Built on the open-source Jmix framework. Source code available on https://github.com/flowset/flowset-control-community[GitHub].

==== Studio
A plugin for https://www.jetbrains.com/idea/[IntelliJ IDEA]. Adds a BPMN/DMN editor directly into the development environment, along with additional capabilities for code generation, navigation, etc.

==== Tasklist
A React-based application template for managing end-user tasks. Source code available on https://github.com/flowset/flowset-tasklist-react-community[GitHub].
